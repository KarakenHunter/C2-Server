# TCP Remote Command Tool

## Overview
This project demonstrates a basic client-server architecture using TCP sockets in Python. It allows a client to send commands to a server, which executes them and returns the output.

The purpose of this project is to understand low-level networking, socket communication, and the fundamentals of remote command execution in a controlled environment.

---

## Features
- TCP-based client-server communication
- Remote command execution
- Simple and lightweight implementation
- Easy to extend for further experimentation
